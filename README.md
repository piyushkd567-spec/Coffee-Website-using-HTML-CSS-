# Coffee-Website-using-HTML-CSS-
A responsive, pixel-perfect e-commerce landing page and footer UI built using pure HTML5 and CSS3, featuring modern Flexbox layouts and component-isolated styling.
A clean, responsive, pixel-perfect e-commerce landing page clone built using pure HTML5 and CSS3. This project focuses on modern flexbox layouts, isolated CSS component styling, and clean typographic structure.

✨ Key Features
100% Pure HTML & CSS: Built entirely from scratch without using heavy external frameworks like Bootstrap or Tailwind.

Flexbox Architecture: Multi-column layout structure using CSS Flexbox for clean alignment across navigation menus and footer blocks.

Component Isolation: Scope-protected CSS rules designed to prevent navbar and footer style leakages.

Responsive Layout: Responsive container styling ensuring legal disclaimers and navigation links scale across different screen sizes.

🛠️ Tech Stack
HTML5: Semantic markup (<ul>, <li>, <p>) for better accessibility and structure.

CSS3: Custom styles utilizing Flexbox, custom positioning, and standard resets.

Font Awesome: Iconography for contact details and navigation icons.
